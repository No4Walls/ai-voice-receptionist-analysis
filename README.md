## 📦 Installation

### Prerequisites
- Node.js 18+ 
- pnpm package manager

### Setup
```bash
# Clone the repository
git clone https://github.com/yourusername/ai-voice-receptionist-analysis.git
cd ai-voice-receptionist-analysis

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

The application will be available at `http://localhost:5173`

## 🚀 Usage
